# Free Publishing Checklist

## Recommended zero-cost route

Publish the ebook in three places:

1. GitHub repository for the source manuscript.
2. GitHub Pages from the `docs` folder for online reading and direct EPUB download.
3. Leanpub with minimum price set to free, if you want a reader-friendly store page and optional support payments.

Optional archival copy:

- Internet Archive, if you want a long-term public archive.

Avoid relying on Amazon KDP for a permanently free ebook. KDP is free to upload to, but Amazon says Kindle ebooks cannot have their list price set to free.

## License options

Choose carefully because Creative Commons licenses are designed to be long-term public permissions.

- `All rights reserved`: free to read, but others cannot redistribute or modify without permission.
- `CC BY 4.0`: others can share and adapt, even commercially, with credit.
- `CC BY-NC 4.0`: others can share and adapt for noncommercial use with credit.
- `CC BY-NC-ND 4.0`: others can share noncommercially with credit, but cannot modify.

For a free learning ebook, a practical choice is usually `CC BY-NC 4.0` if you want people to share it but not sell it.
