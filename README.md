# From Frontend Developer to AI Engineer

Indian-style practical ebook for Angular and React developers who want to upgrade into AI engineering.

## Author

Ashish Valecha  
mrashishvalecha@gmail.com

Ashish Valecha is a developer and practical technology learner who writes for Indian software engineers upgrading their skills for modern product work. This ebook explains AI engineering using familiar Indian daily-life examples, Hinglish explanations, and project-based learning.

## Read and Download

When published with GitHub Pages, readers can:

- Read online from `docs/index.html`
- Download EPUB from `docs/frontend-to-ai-engineer-indian-style.epub`
- Read the source manuscript in `frontend-to-ai-engineer-indian-style.md`

## What This Book Covers

- Why frontend developers can become AI engineers
- Python basics for JavaScript developers
- LLMs and prompt engineering
- Embeddings and vector search
- RAG systems
- AI agents
- Evaluation and safety
- Deployment, cost, and monitoring
- Portfolio projects and interview preparation
